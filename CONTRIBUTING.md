## ğŸš€ Quick Start Guide

### 1. Fork & Clone
Fork the repo on GitHub and clone it to your local machine:

```bash
git clone https://github.com/YOUR-USERNAME/ValuationAI.git
cd ValuationAI
```

### 2. Install Dependencies
Make sure you have Node.js (v18+) installed, then run:

```bash
npm install
```

### 3. Create a Branch
Always work on a new branch for your changes:

```bash
git checkout -b feature/amazing-feature
# or
git checkout -b fix/critical-bug
```

### 4. Develop & Test
Start the development server and make your magic happen:

```bash
npm run dev
```

### 5. Commit Changes
Follow our commit message convention (Conventional Commits):

```bash
git commit -m "feat: add new valuation metric"
git commit -m "fix: resolve calculation error in safari"
```

### 6. Push & Pull Request
Push to your fork and submit a Pull Request:

```bash
git push origin feature/amazing-feature
```

## ğŸ“ Coding Standards

To ensure code quality and consistency, please adhere to the following:

*   **Style Guide:** We use ESLint and Prettier. Run `npm run lint` before committing.
*   **File Naming:** Use `PascalCase` for components (e.g., `ValuationCard.jsx`) and `camelCase` for utilities (e.g., `calculateMetrics.js`).
*   **Comments:** Write meaningful comments for complex logic. Explain *why*, not just *what*.
*   **Hooks:** Keep custom hooks in `src/hooks/` and prefix them with `use`.
